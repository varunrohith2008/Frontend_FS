JavaScript (JS)


Concepts ES5 ES6
Variable Declaration => Uses var (function, can be redeclared) =>Uses let and const (Block Scoped - Cannot be reclared)
Functions =>Defined using function keyword =>Introduced arrow functions () => {} with lexical this
Strings =>String concatenation using + =>Introduced template literals using backticks `Hello ${name}`
Objects =>Must specify key-value pairs explicitly =>Supports object shorthand, computed property names, and method shorthand
Classes =>Implemented via constructor functions & prototypes =>Introduced class syntax with constructor, extends, super
Modules =>No native module system (used IIFE/CommonJS require) =>Introduced native import and export statements
Parameters =>Set manually inside function body =>Supports default parameter values in function definitions
Destructuring =>Manual extraction of values from arrays/objects =>Introduced destructuring assignment [a, b] = arr, {x, y} = obj
Promises/Async =>Uses callbacks for async code =>Introduced Promises, later enabling (async/await)
Loops/Iteration =>for, for-in loops =>Added for...of, iterators, and generators (function)
Scope =>Function scope (via var) =>Function scope (via var)
Hoisting =>Variables hoisted and initialized as undefined =>Variables hoisted but not initialized (Temporal Dead Zone)
Type of Syntax =>Verbose, older syntax =>Cleaner, more readable, modern syntax
Browser Support =>Supported by all browsers =>Supported by modern browsers (can use Babel for older ones)
